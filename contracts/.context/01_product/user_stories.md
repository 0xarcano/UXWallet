# User Stories

- As a user, I want to deposit my ETH into a secure vault so it can be used for yield-bearing solver activities.
- As a user, I want to withdraw my funds instantly if I provide a valid off-chain state proof.
- As a protocol, I want to rebalance assets across chains without moving physical funds on-chain.
- As a user, I want to revoke my session key instantly on-chain in case of a backend breach.
- As a protocol, I want to use the Treasury to subsidize a withdrawal fee if local liquidity is missing.
- As a developer, I want to ensure the ExecutionGuard prevents assets from leaving the vault without an atomic counterpart.
