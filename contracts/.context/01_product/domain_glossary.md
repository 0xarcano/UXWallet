# Domain Glossary

| Term | Definition |
|------|------------|
| **UXVault** | The on-chain vault holding user assets. |
| **Adjudicator** | The contract responsible for verifying Nitrolite state updates. |
| **Checkpoint** | An on-chain proof of off-chain state. |
| **Force Withdrawal** | An escape hatch for users to claim funds without protocol permission. |
