# Domain Glossary

| Term | Definition |
|------|------------|
| **UXVault** | The on-chain vault holding user assets. |
| **Checkpoint** | An on-chain proof of off-chain state. |
| **Force Withdrawal** | An escape hatch for users to claim funds without protocol permission. |
| **SessionKeyRegistry** | Mapping of Owner => SessionKey with expiry and permission bits. |
| **Execution Guard** | Logic ensuring physical funds release is coupled with on-chain or off-chain proof of arrival. |
| **Sponsorship Module** | Logic for the treasury to pay fees to external solvers during "Hybrid Exits." |
| **Adjudicator** | The ERC-7824 contract verifying off-chain state updates. |
