# Domain Glossary

| Term | Definition |
|------|------------|
| **ClearNode** | The server responsible for co-signing off-chain state updates. |
| **Intent Listener** | A service that scans the LI.FI Marketplace for eligible orders. |
| **Settlement Engine** | The logic that determines when to batch off-chain updates to the VXVault. |
| **Virtual Ledger** | The off-chain database tracking real-time user balance. |
