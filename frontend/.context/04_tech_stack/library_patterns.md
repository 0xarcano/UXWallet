# Library Patterns

| Library | Usage |
|---------|-------|
| **lif-rust API** | Use REST API for LI.FI quote fetching and ERC-7683 order encoding during "Unification" deposit and "Hybrid Exit" flows. |
| **Yellow** | Use `@erc7824/nitrolite` for state channel signatures. |
| **EIP-712** | Use for structured data signing during session key delegation. |
| **HTTP Client** | Use fetch/axios to communicate with lif-rust microservice. |
