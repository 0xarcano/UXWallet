# Coding Style

| Area | Rule |
|------|------|
| **Framework** | Next.js 14+ (App Router), TypeScript. |
| **Styling** | Tailwind CSS with Shadcn UI components. |
| **Icons** | Lucide-react. |
| **UX Rule** | "One-Click Approval" philosophy. No manual chain-switching for the user after initial deposit. |
