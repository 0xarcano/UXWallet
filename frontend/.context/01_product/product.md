# Product

| Attribute | Description |
|-----------|-------------|
| **Name** | Flywheel Wallet (Mobile App). |
| **Purpose** | Non-custodial Aggregated Liquidity Wallet. Users delegate assets to the pool and grant the Flywheel Solver permission (Session Key via EIP-712); view unified balance across chains; send (gasless L3 + cross-chain); withdraw with Fast Exit Guarantee. Real-time updates via ClearNode WebSocket. |
| **Target Users** | Web3 medium and advanced users. |
| **Platforms** | iOS (primary), Android (primary), Web (secondary). |
